# k8s-tests

Personal Kubernetes test repository used in my homelab.

This repo contains all the manifests, experiments, and configurations I create while following lectures and hands-on labs on KodeKloud. It acts as a sandbox for learning, testing concepts, and validating behaviors in a real cluster environment.

## Purpose

- Practice Kubernetes concepts in a real environment
- Validate lab exercises outside of the course platform
- Keep a history of working configurations and experiments
- Build muscle memory through repetition and iteration

## Environment

- Local homelab cluster (Minikube)
- Multiple nodes setup for testing scheduling and networking
- Used alongside KodeKloud courses and labs

## What you will find here

- Pod, Deployment, Service definitions
- Volumes and Persistent Volume tests
- Networking and DNS experiments
- RBAC and security configurations
- Scheduling concepts like taints, tolerations, and affinity
- Misc lab exercises and troubleshooting scenarios

## Notes

This repository is not meant to be production-ready.  
It is a learning space, things may be incomplete, experimental, or intentionally broken for testing purposes.
